特別課題３　パズル

１．５＊５のフィールドを用いて、下記のルールのパズルを作れ。

0とXはこまの表裏を表す。
XYで座標を入力する。
入力した座標と、その上下左右が裏返る。
すべてこまを表(0)の状態から初めて、すべて裏(X)になったらパズル終了。
座標入力で「zz」が入力されたらパズルを中止する。

  a b c d e
A 0 0 0 0 0
B 0 0 0 0 0
C 0 0 0 0 0
D 0 0 0 0 0
E 0 0 0 0 0
>座標入力：bB

  a b c d e
A 0 X 0 0 0
B X X X 0 0
C 0 X 0 0 0
D 0 0 0 0 0
E 0 0 0 0 0
>座標入力：cC

  a b c d e
A 0 X 0 0 0
B X X 0 0 0

D 0 0 X 0 0
